# 🛡️ Cybersecurity Portfolio

This repository contains hands-on cybersecurity labs focused on SOC (Security Operations Center) skills, including detection, analysis, and incident investigation using real tools such as Wazuh SIEM.

## 🔍 Focus Areas

- Security Monitoring
- Threat Detection
- Log Analysis
- SIEM (Wazuh)
- MITRE ATT&CK Mapping

## 🧪 SOC Lab

This lab environment simulates real-world attack scenarios using a segmented virtual network and monitored endpoints.

### 📁 Labs

- Lab 01 – Windows Failed Logon Detection
- Lab 02 – SSH Brute Force Detection

---

## 🚀 Objective

To develop practical, real-world SOC analyst skills through hands-on detection and analysis of simulated cyberattacks.
